Stellio is an NGSI-LD compliant context broker developed by EGM. NGSI-LD is an Open API and Datamodel specification for context management `published by ETSI <https://www.etsi.org/deliver/etsi_gs/CIM/001_099/009/01.02.02_60/gs_CIM009v010202p.pdf>`__

This project is part of `FIWARE <https://www.fiware.org/>`__. For more information check the FIWARE Catalogue entry for `Core Context <https://github.com/Fiware/catalogue/tree/master/core>`__.
